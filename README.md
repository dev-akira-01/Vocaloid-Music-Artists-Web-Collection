#  Vocaloid Producer Web Collection

A multi-page **semantic HTML project** dedicated to documenting influential modern **Vocaloid producers**.

This repository is being developed as a long-form HTML practice project focused on **clean document structure,accessibility, and information**. Each page aims to function as a detailed artist reference page rather than a simple static webpage.(I mean they are static pages without JS, but will try my best to make it as clean as possible)

The project is being expanded **incrementally**, with new producer pages written and structured over time.

---

##  Project Overview

This project is designed as a **mini wiki website** for Vocaloid music producers.  
Each artist page explores their career, musical style, notable works, and impact on the Vocaloid community.

Rather than focusing purely on visual design, the goal of the project is to prioritize:

- **Well-structured HTML**
- **Content-rich documentation pages**

##  HTML Concepts Practiced

This project intentionally focuses on **semantic-first HTML design**.

### Semantic Layout Elements

- `<header>`
- `<main>`
- `<article>`
- `<section>`
- `<nav>`
- `<footer>`
- `<aside>`

### Structured Content Elements

- `<figure>` and `<figcaption>`
- `<table>` for structured information
- `<blockquote>` for citations
- `<details>` and `<summary>` for expandable content
- `<abbr>` for terminology
- `<time>` for temporal metadata

### Accessibility Considerations

- Descriptive `alt` attributes
- `aria-label` for navigation landmarks
- Structured navigation sections
